🚀 What is this?

A fully automated YouTube Shorts pipeline that turns a one-line topic into a published video in minutes.

Pipeline Flow:
Research → Script → AI Visuals → Voiceover → Captions → Music → Upload

No video editing skills needed. Just provide a topic and let AI do the rest.

📋 Pipeline Stages

Stage What Happens Tools Used
Draft DuckDuckGo research → Claude script → b-roll prompts, thumbnail prompt Claude, DuckDuckGo
Produce Gemini Imagen b-roll → ElevenLabs voiceover → Whisper captions → music → ffmpeg assembly Gemini, ElevenLabs, Whisper, ffmpeg

Anti-Hallucination Gate: Claude only uses names and facts from live DuckDuckGo research — not training data.

⚙️ Setup Guide

1. Install dependencies

pip install -r requirements.txt

2. Run pipeline (first run launches setup wizard)

cd youtube-shorts-pipeline python -m pipeline run --news "your topic here" --dry-run

3. Set up YouTube OAuth (for upload)

python scripts/setup_youtube_oauth.py

Key Commands

# Draft stage only (test script) python -m pipeline draft --news "your topic" # Auto-pick from trending python -m pipeline draft --discover --auto-pick # Produce video from draft python -m pipeline produce --draft ~/.youtube-shorts-pipeline/drafts/.json # Full pipeline python -m pipeline run --news "your topic" # List trending topics python -m pipeline topics

Required API Keys (in config.json)

ANTHROPIC_API_KEY — Required (Claude for script)

GEMINI_API_KEY Required (Imagen for visuals)

ELEVENLABS_API_KEY Optional (TTS, macOS say fallback)

🔒 Security Features

Credential Storage: config.json and youtube_token.json created with 0600 permissions (owner-only)

API Key Transmission: Gemini key sent via x-goog-api-key header, not URL query parameter

Error Handling: API error messages sanitized to never include credentials

Upload Privacy: Videos uploaded as private by default

OAuth Scope: Minimum scopes needed (youtube.upload + youtube.force-ssl)

Important: Never commit config.json or youtube_token.json to git — they're in .gitignore
