🤖 What is K-Dense BYOK?

K-Dense BYOK (Bring Your Own Keys) is an open-source desktop app that gives you an AI research assistant called Kady.

💡 The Vision: Instead of paying subscriptions to single AI companies, you use your own API keys. You control which models to use, how much you spend, and where your data goes.

Built for scientists, analysts, and curious people who want a powerful AI workspace without being locked into one provider.

🚀 Setup Guide

1. Clone the repo

git clone https://github.com/K-Dense-AI/k-dense-byok.git cd k-dense-byok

2. Create .env file

Create kady_agent/.env with: # Required OPENROUTER_API_KEY=paste-your-key-here # Optional (for web search) PARALLEL_API_KEY=paste-your-key-here # Leave as-is GOOGLE_GEMINI_BASE_URL=http://localhost:4000 GEMINI_API_KEY=sk-litellm-local DEFAULT_AGENT_MODEL=openrouter/google/gemini-3.1-pro-preview

3. Run the app

chmod +x start.sh ./start.sh
First run installs Python packages, Node.js, Gemini CLI, and downloads scientific skills — may take a few minutes.

4. Access the app

Browser opens to http://localhost:3000

🏗️ How it Works

Three Services Run Locally

Frontend (port 3000) — Web interface, chat, file browser

Backend (port 8000) — Runs Kady and coordinates expert tasks

LiteLLM proxy (port 4000) — Routes AI requests to chosen model via OpenRouter

When you send a message:

  1. Kady reads your message
  2. Decides whether to answer directly or delegate to an expert
  3. Uses tools (web search, file operations, scientific databases)
  4. Streams response back to you
