What is Codebase to Course?

A Claude Code skill that transforms any GitHub repository into a beautiful, self-contained HTML course. Perfect for "vibe coders" — people who build software using AI tools without traditional CS education.

🎯 Who is this for?
You build things with AI coding tools, but sometimes don't fully understand how the code works. This tool teaches you — not by lecturing, but by tracing what happens when you use the app.

How It Works

1. Point at a Repo

Give Claude Code a GitHub repository URL. It analyzes the entire codebase structure.

2. Get a Course

Out comes a single HTML file — no dependencies, works offline, teaches you how the code works.

3. Learn by Doing

Instead of memorizing concepts, you experience the app working first — then understand how it works.

📖 How to Use It

Installation

Trigger Commands

"Turn this codebase into an interactive course" "Turn this into a course" "Explain this codebase interactively" "Make a course from this project" "Teach me how this code works" "Interactive tutorial from this code"

File Structure

codebase-to-course/ ├── SKILL.md # Main skill instructions └── references/ ├── design-system.md # CSS, typography, colors └── interactive-elements.md # Quiz & animation patterns
