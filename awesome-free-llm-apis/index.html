📚 What is this?

A curated list of free LLM APIs that actually work — no trial periods, no credit card required, no time limits.

💡 Why it matters: You can build AI apps, test ideas, and prototype without spending a dime. Perfect for hobbyists, students, and bootstrapped developers.

Two Types of Providers:

Direct Providers — Companies that train their own models (Cohere, Google, Mistral)
Third Party — Platforms hosting open-weight models (Groq, Hugging Face, Ollama)

💎 Pro Tips

🔑 Get API Keys: Click each provider's name to go to their API key page. Some require account creation but no payment.
🔄 OpenAI SDK Compatible: Most work with the standard OpenAI Python library — just change the base URL and API key!
⚠️ Watch Out For:
  • Some providers not available in EU/UK
  • Rate limits vary by model (larger models = stricter limits)
  • Free tier can be revoked — don't rely on it for production
🚀 Best for Beginners:
  1. Groq — Fastest inference, generous limits
  2. Cohere — Reliable, great models
  3. OpenRouter — Best model variety
